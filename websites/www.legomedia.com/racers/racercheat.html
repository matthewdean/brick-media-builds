The LEGO Racers cheat code for December 1999 turns your car into a jet-powered hovercraft! This cheat code works with all versions of LEGO Racers on all game platforms.

To get the cheat code to work, follow these instructions:

  1. Start LEGO Racers.
  2. Enter build mode.
  3. Create a new driver (or edit an existing one).
  4. Go to make license.
  5. Type in FLYSKYHGH as the driver's name.
  6. Start a single or versus race.
