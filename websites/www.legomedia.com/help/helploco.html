Q. Why is the Stationmaster speaking a foreign language?
 A. The Station Master and the characters in the game are not speaking in any specific language. The language choice you are asked to make at installation is for the text present during the installation procedure and whilst playing.

Q. Why do I get the error message "component move data error" when I try to install LEGO Loco?
 A. Usually this is associated with a failed attempt at installation and then attempting to reinstall. Once you have installed LEGO Loco you no longer need to use the CD. The program runs entirely off your hard drive. If you have attempted multiple installs then uninstall LEGO Loco from ADD/REMOVE PROGRAMS in the CONTROL PANEL and reinstall it only once.

Q. LEGO Loco does not work properly on my computer. Do I meet the minimum system requirements to run the software?
 A. The minimum system requirements are as follows:
  • Windows 95/98
  • Pentium 133MHz or equivalent
  • 32MB RAM
  • 4x speed CD-ROM drive
  • 50MB of free hard drive space after install (205MB needed for full install--without the video component takes 103MB)
  • 1MB video card
  • Mouse
Q. When I start LEGO Logo I get an error message about colours and screen size.
 A. You need to change the colour depth and resolution of your desktop. To change the display settings:
    Click on Start
    Click on Settings
    Click on Control Panel
    Double click on Display
    Select the tab marked Settings

The colour settings must be High Colour (16-bit) or more. Also make sure that the screen size is set to at least 800 x 600.

Q. When playing or loading the game crashes and reports an Invalid Page Fault in Kernel32.dll.
 A. First close all open programs and re-boot the computer. Then try to run the program again. If the problem persists we recommend downloading the Kernel32 patch from the Microsoft web site which can be found here.
If the problem persists or you are running Windows98 please contact your regional technical support, details of which can be found here.
