An upgrade patch is available for the English language versions of LEGO Chess (i.e. the US and UK versions) which resolves the following problems (please note that users of other language versions will not encounter these problems):

  • The game crashes when loading a Staunton board saved game.
  • The game sometimes crashes when switching between menus.
  • There are incorrect responses to some moves in the tutorial.

    The patch is available for download from here.
    Double click on the downloaded file. It will create a folder on your Windows desktop called LEGO Chess Update, which contains a file called Upgrade206b_int.exe.
    Double click on this file to patch LEGO Chess.


    Q. LEGO Chess does not load normally - it only works if I load the LEGO Chess Configuration program first and then click on PLAY LEGO CHESS.
     A. Unfortunately, this is a known bug which is specific to a small number of PCs. The developers are currently working on a new patch that will fix this problem. In the meantime, please load LEGO Chess via the Configuration program.

    Q. Why is my mouse moving sporadically?
     A. Typically this is associated with the wrong video driver being selected. First go into the CHESS CONFIGURATION icon and change the driver to KRISALIS SOFTWARE DRIVER. Also lower the resolution settings to 512x384x16, 640x200x16, 640x240x16, 640x400x16 or 640x480x16. Then CLICK on PLAY LEGO CHESS.

    Q. How can I improve the performance of the game?
     A. The first thing to do is a CTRL-ALT-DEL and END TASK on all programs except EXPLORER and SYSTRAY. Also empty your TEMP folder in the WINDOWS directory. These files are only temporary so if they are needed again they will be recreated. Also to improve the performance you can add a graphics accelerator card with a minimum of 4MB RAM.


    Q. When I play with the Staunton board parts of the screen are corrupted.
     A. You must turn off reflections within the settings menu. This stops the problem from occurring.
